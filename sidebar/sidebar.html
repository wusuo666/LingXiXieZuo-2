    AI绘图助手

    我可以帮助你创建画布、添加形状、添加文本或查看画布信息等。请在下方输入你的问题或选择上方工具。

    可用工具列表

    ×

    创建画布 (createCanvas)

    创建新的Excalidraw画布，可选择模板

    name: 画布名称（不需要包含.excalidraw扩展名）
    template: 可选的模板名称，可用模板: 空白画布, 基础图形, 流程图, 思维导图, 组织结构图
    例如: 创建一个名为flowchart的画布，使用流程图模板

    列出画布 (listCanvases)

    列出所有已创建的Excalidraw画布

    例如: 列出所有画布

    导出画布 (exportCanvas)

    将Excalidraw画布导出为SVG格式

    name: 画布名称
    format: 导出格式: svg
    withBackground: 是否包含背景（默认为true）
    withDarkMode: 是否使用暗色模式（默认为false）
    exportScale: 导出缩放比例（默认为1）
    例如: 将flowchart画布导出为SVG格式，无背景

    添加形状 (addShape)

    向Excalidraw画布添加基本形状

    name: 画布名称
    shapeType: 形状类型: rectangle, ellipse, diamond, line, arrow, text
    x: X坐标位置
    y: Y坐标位置
    color: 颜色代码，如 #000000 或 #ff0000
    例如: 在flowchart画布上添加一个矩形，位置(100,100)，颜色蓝色

    添加文本 (addText)

    向Excalidraw画布添加文本

    name: 画布名称
    text: 文本内容
    x: X坐标位置
    y: Y坐标位置
    color: 文本颜色（可选，默认为黑色）
    fontSize: 字体大小（可选，默认为20）
    例如: 在flowchart画布上添加文本"开始"，位置(150,150)

    获取画布详情 (getCanvasDetails)

    获取Excalidraw画布的详细信息

    name: 画布名称
    例如: 获取flowchart画布的详细信息

    导入库 (importLibrary)

    导入Excalidraw公共库或从URL导入库

    libraryUrl: 库URL或识别符（如"rocket"、"charts"等公共库ID或完整URL）
    canvasName: 可选：要导入到的画布名称。如不提供，将作为工作区库导入
    例如: 导入"rocket"库到flowchart画布

    删除元素 (deleteElement)

    从Excalidraw画布中删除指定元素

    name: 画布名称
    elementId: 要删除的元素ID
    例如: 从flowchart画布中删除ID为"rectangle1"的元素

    画框工具 (createFrame)

    在画布中创建一个框架用于分组元素

    name: 画布名称
    x: 框架X坐标位置
    y: 框架Y坐标位置
    width: 框架宽度
    height: 框架高度
    label: 框架标签文本（默认为"框架"）
    color: 框架颜色代码（默认为蓝色）
    例如: 在flowchart画布上创建一个标签为"模块A"的框架，位置(200, 200)，尺寸300x200

    嵌入网页 (embedWebpage)

    在画布中嵌入网页链接

    name: 画布名称
    url: 要嵌入的网页URL
    x: X坐标位置
    y: Y坐标位置
    width: 宽度（默认为320）
    height: 高度（默认为180）
    例如: 在flowchart画布上嵌入网页 https://github.com，位置(100, 300)

    调整元素样式 (updateElementStyle)

    调整画布中元素的样式属性

    name: 画布名称
    elementId: 要调整样式的元素ID
    styleOptions: 样式选项对象，可包含以下属性:
    strokeColor: 线条颜色
    backgroundColor: 背景颜色
    fillStyle: 填充样式(solid, hachure, cross-hatch等)
    strokeWidth: 线条宽度
    strokeStyle: 线条样式(solid, dashed等)
    opacity: 不透明度(0-100)
    fontSize: 字体大小(文本元素)
    例如: 更新flowchart画布中ID为"rect1"的矩形样式，将背景色改为红色，线条宽度改为2
